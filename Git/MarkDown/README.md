### MarkDown이란?

확장자가 '.md' 로 되어 있는 파일 이고 주로 git 업로드 시 프로젝트 설명에 자주 사용된다. 
web에서 소비되는 워드 라고 생각 하면 편하다. 
HTML로 변환이 가능 하기 때문에, 문법을 사용하여 제목, 볼드체, 이탤릭, 목록, 표 등을 만들 수 있다. 

VS code / Xcode / intelliJ 등 많은 IDE에서 .mb파일을 지원한다. 
가장 편한 툴이 mb파일을 지원한다면 사용해 볼 것 



### MarkDown 기초 문법

뛰어쓰기에 매우 매우 매우 민감하니, 띄어쓰기를 잘 하는것이 **중요**

#### **제목**

'Header' : 제목, 문단별 제목, 소제목 
    글의 큰 틀, 구조를 잡을 때 사용한다. 
    html과 동일하게 6단계를 제공한다.  

        # 제목 -1-
        ## 제목 -2-  
        ### 제목 -3-
        #### 제목 -4-
        ##### 제목 -5-
        ###### 제목 -6-
        
    
# 제목 -1-
## 제목 -2-  
### 제목 -3-
#### 제목 -4-
##### 제목 -5-
###### 제목 -6-

~~~

1. 목록은

4. 이상하게 
5. 나열해도 
10. OK!

#### **테이블(표)**  
    ' | ' 와 ' ---' 으로 표를 만들 수 있다  
    아래와 같이 입력 할 수 있다.  
    매 row 마다 '|' 로 분리하고 주제 행의 아래 '-'를 그어 표를 만들 수 있다. 

- 기본 표 
    ~~~
    |제목|제목|제목|
    |---|---|---|
    |테스트1|테스트2|테스트3|
    |테스트1|테스트2|테스트3|
    |테스트1|테스트2|테스트3|
        
    ~~~
  
    |제목|제목|제목|
    |------|---|---|
    |테스트1|테스트2|테스트3|
    |테스트1|테스트2|테스트3|
    |테스트1|테스트2|테스트3|

- 정렬
    
    ~~~
    |제목|제목|제목|
    |:---|:---:|---:|
    |테스트1|테스트2|테스트3|
    ~~~
  
    |제목|제목|제목|
    |:---------------------|:-----------------:|-------------------:|
    |왼쪽정렬|중앙정렬|오른쪽정렬|
    
- 강조 및 줄바꿈
  
    ~~~
    |제목|제목|제목|
    |------|---|---|
    | 강조 | **볼드** | *이태릭* |
    | 색상 | <span style="color=red">빨간색</span> | <span style="color=blue">빨간색</span>|
    | 줄바꿈 | 여기서 줄 바꿈은 허용이 안되요  | 하지만 <br> 이건 가능!!|
    ~~~
    
    |제목|제목|제목|
    |------|---|---|
    | 강조 | **볼드** | *이태릭* |
    | 색상 | <span style="color=red">빨간색</span> | <span style="color=blue">파란색</span>|
    | 줄바꿈 | 여기서 줄 바꿈은 허용이 안되요  | 하지만 <br> 이건 가능!!|

#### **코드블럭(Code Block)**  

'~~~' / '```' 를 사용하여 코드블럭을 만들 수 있다.
참고로 ` 은 맥의 경우 키보드가 영어로 되어 있을때 ₩를 누르면 `가 출력된다.

~~~
 '~~~' / '```' 를 사용하여 코드블럭을 만들 수 있다. 
 참고로 ` 은 맥의 경우 키보드가 영어로 되어 있을때 ₩를 누르면 `가 출력된다. 
~~~

특정언어를 명시하면 문법에 따른 색상이 적용된다. 

    ``` java
        int val = 10;
        if(val + 5 == 10)
            // 참
    ```

``` java
    int val = 10;
    if(val + 5 == 10)
        // 참
```

#### **링크 (Link)**

    - *외부 링크* 
    
    ```
       인라인 링크 :   
        [인라인 링크](http://www.google.com)  
        
        참조 링크 : [참조 링크][1]  // 번호를 맞추면 링크가 연결

    ```
    
    인라인 링크 : [인라인 링크](http://www.google.com)  
    
    참조 링크 : [참조 링크][1]  
    
    [1]: http://www.google.com/ "링크제목1"

    [링크](#id)
