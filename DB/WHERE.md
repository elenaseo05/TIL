# **WHERE**
---------

##  조건문

문법
```sql 
SELECT 출력할 컬럼명
FROM 테이블명
WHERE 출력하기 원하는 특정조건
```


- 비교연산자 (숫자)

- 숫자, 문자, 날짜 데이터에 사용가능
    =    값이 동일한 행을 찾는다.
    \>    보다 크다. 초과
    <    보다 작다. 미만
    \>=    크거나 같다. 이상
    <=    작거나 같다. 이하
    !=, <>, ^=    같지 않다.


- 비교연산자 (문자열)
    -  ‘ ‘ 작은따옴표 = 문자열 
    -  문자열 조회
    -  대소문자 구분


- 비교연산자 (날짜)
    - 날짜 조회
